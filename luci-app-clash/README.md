 ## Install
- Upload ipk file to tmp folder

- cd /tmp
- opkg update
- opkg install luci-app-clash_1.7.3.1_all.ipk  
- opkg install luci-app-clash_1.7.3.1_all.ipk --force-depends

 ## Uninstall
- opkg remove luci-app-clash 
- opkg remove luci-app-clash --force-remove

## Features
- Suport Subscription Config
- Support Config Upload
- Support Create Config
- GeoIP Database Update
- TProxy UDP
- IP Query
- DNS Forwarding
- Support Trojan
- Support SSR
- Support V2ray
- Ping Custom Proxy Servers
- Tun Support
- Access Control
- Support Provider,
- Game Rules 
- Auto Restore Config
